Letter Board is a tool to support excersizes for improving distant vision based on description in Better Vision Now: Improve Your Sight with the Renowned Bates Method by Clara A. Hackett and Lawrence Galton.

Letter Board is designed to run on a large screen, such as TV or a large computer monitor. It comes with a companion application, Hand Board, designed to run on a handheld device such as mobile phone or a tablet.

Begin by setting up the Letter Board: select font, adjust letter size as well as vertical and horizontal letter spacing so that letters take up the whole screen. Position yourself in front of the screen at a distance where letters are slightly blurred (or/and adjust letter size).

Open Hand Board on a phone or a tablet: generate QR code for Hand Board matching current letter layout by tapping QR code icon or type q.

Scan QR code using a bar code scanner app. Alternatively, open generated Hand Board URL, copy it and send it to your handheld device via email or instant message. Open Hand Board URL in a browser.

Tap play button to start the excersize. Look briefly at the two letters (large and small, same character) at the top of the Hand Board. Note which one is black (large or small) and find the same letter of matching size on the distant board. Then find corresponding location on the Hand Board and tap it to place the letter. If placed correctly, next letter will be displayed at the top. Continue until all letters are placed.
